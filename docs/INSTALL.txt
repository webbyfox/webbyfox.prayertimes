webbyfox.prayertimes Installation
---------------------------------

To install webbyfox.prayertimes using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``webbyfox.prayertimes`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        webbyfox.prayertimes
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
